# 03 / 成果物テンプレート集

各フェーズで Claude が埋めて出力する**テンプレート**。
すべて **要件ID（FR/NFR/CMP/INT）** でトレーサビリティを維持する。

## フェーズ共通：完了の定義（DoD）チェック

- [ ] 成果物がテンプレートの章立てを満たす
- [ ] すべての主要項目に要件IDが紐づく（トレーサビリティ）
- [ ] 未確定は仮説台帳に、確定は決定台帳に記録済み
- [ ] 法令・個人情報該当箇所に専門家確認フラグ
- [ ] MVPと将来が明確に分離されている
- [ ] ゲートチェックリストを満たし、ユーザー承認を取得した
