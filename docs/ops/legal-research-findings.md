## A. 所得税速算表・令和7年度改正 (issue #20)

### A-1. 速算表ブラケット (令和6・令和7分 不変)

出典: 国税庁 No.2260 所得税の税率
<https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2260.htm>

| 課税所得(円) | 税率 | 控除額(円) |
|---|---|---|
| 1,000 〜 1,949,000 | 5% | 0 |
| 1,950,000 〜 3,299,000 | 10% | 97,500 |
| 3,300,000 〜 6,949,000 | 20% | 427,500 |
| 6,950,000 〜 8,999,000 | 23% | 636,000 |
| 9,000,000 〜 17,999,000 | 33% | 1,536,000 |
| 18,000,000 〜 39,999,000 | 40% | 2,796,000 |
| 40,000,000 以上 | 45% | 4,796,000 |

**確認事項**:
- 境界値は円単位で正確に管理すること(万円丸め禁止)。
- 速算表だけでは最終所得税額にならない: 復興特別所得税(基準所得税額×2.1%、
  2013–2037年適用)を別途上乗せ。
- 令和7年以後の超高所得者サーチャージ(課税所得330百万円超部分)は別建て(未実装)。

**コードへの反映**: `backend/internal/yearend/service.go` の `computeIncomeTax` 関数の
境界値を一次情報と一致するよう修正済み(1,949,000 / 3,299,000 / 6,949,000 /
8,999,000 / 17,999,000 / 39,999,000)。

### A-2. 令和7年度税制改正 (施行=令和7年12月1日、令和7年分以後適用)

施行日: 令和7(2025)年12月1日。本日(2026-06-04)は既に適用済み年分。

| 改正項目 | 令和6以前 | 令和7以後 | 出典 |

| 基礎控除 (合計所得2,400万以下) | 48万円(一律) | 段階制(下記) | 国税庁 No.1199 / 特設 |
| 給与所得控除の最低保障額 | 55万円 | 65万円 | 国税庁 No.1410 |
| 扶養親族・同一生計配偶者の合計所得要件 | 48万円 | 58万円 | 国税庁 No.1191 |

**基礎控除の段階制(令和7年分)**:

出典: 国税庁 https://www.nta.go.jp/users/gensen/2025kiso/index.htm
/ 国税庁 No.1199 https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1199.htm

| 合計所得金額 | 基礎控除額 |
|---|---|
| 132万円以下 | 95万円 |
| 132万円超 336万円以下 | 88万円 |
| 336万円超 489万円以下 | 68万円 |
| 489万円超 655万円以下 | 63万円 |
| 655万円超 2,350万円以下 | 58万円 |
| 2,350万円超 2,400万円以下 | 48万円(漸減) |
| 2,400万円超 2,450万円以下 | 32万円 |
| 2,450万円超 2,500万円以下 | 16万円 |
| 2,500万円超 | 0円 |

> **時限措置**: 132万円超655万円以下の中間層への上乗せは**令和7・8年分のみ**。
> 令和9年分以後は一律58万円に戻る(令和6以前の480,000円とは異なる点に注意)。

**配偶者控除額(令和6・7とも不変)**:

出典: 国税庁 No.1191 https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1191.htm

| 本人合計所得 | 一般配偶者 | 老人配偶者(70歳以上) |
|---|---|---|
| 900万円以下 | 38万円 | 48万円 |
| 900万超950万以下 | 26万円 | 32万円 |
| 950万超1,000万以下 | 13万円 | 16万円 |
| 1,000万超 | 不適用 | 不適用 |

**生命保険料控除(所得税、令和6・7とも不変)**:

出典: 国税庁 No.1140 https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1140.htm

- 新契約(H24.1.1以後締結): 一般・医療・個人年金の3区分 各最高4万円
  - 支払額≤2万: 全額
  - 2万超4万以下: ×1/2 + 1万
  - 4万超8万以下: ×1/4 + 2万
  - 8万超: 4万円
- 旧契約(H23.12.31以前): 一般・個人年金の2区分 各最高5万円
  - 支払額≤2.5万: 全額
  - 2.5万超5万以下: ×1/2 + 1.25万
  - 5万超10万以下: ×1/4 + 2.5万
  - 10万超: 5万円
- 新旧通算の合計適用限度: **12万円**
- **住民税は計算式・上限(合計7万円)が異なる。混同しないこと。**

**コードへの反映**: 令和7年度改正は年度依存ロジックを要するため、今回は
`computeIncomeTax` に出典付き TODO コメントを追記し、完全実装を別 issue に委ねる。

## B. 労働基準法 記録保存期間 (issue #18)

出典:
- 厚生労働省 <https://www.mhlw.go.jp/content/000617980.pdf>
- 厚生労働省スタートアップ労働条件 <https://www.startup-roudou.mhlw.go.jp/qa/zigyonushi/syuugyoukisoku/q6.html>
- e-Gov 労働基準法 <https://laws.e-gov.go.jp/law/322AC0000000049>

### 確認済み事実

| 根拠 | 保存期間 | 備考 |
|---|---|---|
| 労基法第109条(令和2年4月1日改正後) | **5年**(原則) | 現行条文の規定値 |
| 労基法附則第143条第1項の経過措置 | **3年**(当面) | 「当分の間」3年とみなす。2026年時点有効、廃止時期未定 |

対象書類: 労働者名簿・賃金台帳・出勤簿・雇入/解雇/災害補償/賃金に関する書類等。

### 設計方針への反映

- 本システムのデフォルト保持期間は**3年**とする(経過措置に対応)。
- 閾値は設定値(configuration)で管理し、将来の5年化(経過措置廃止後)に対応可能な
  設計とする。この設計方針を `retention-service-account.md` に追記済み。

## C. e-Gov 36協定電子申請・電子署名要否 (issue #21)

出典:
- 厚生労働省 36協定電子申請案内 <https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000184033.html>
- 厚生労働省 電子申請の手引き <https://www.mhlw.go.jp/content/11200000/000919894.pdf>
- 本社一括届出 <https://www.mhlw.go.jp/new-info/kobetu/roudou/gyousei/kantoku/dl/130419-1a.pdf>

### 確認済み事実

1. **電子申請の可否**: 36協定(様式第9号系)は e-Gov 電子申請
   (https://shinsei.e-gov.go.jp/)で届出可能。
2. **電子署名・電子証明書の要否**: **令和3(2021)年4月1日以降、不要**。
   デジタル改革関連法により廃止された。
3. **申請単位**: 各事業場単位・本社一括(同一内容要件あり)・医師・建設業等の
   専用手続きが存在する。
4. **本社一括届出の同一内容要件**:
   - 2025年3月31日の労働条件ポータル新チャネル: 事業場間同一内容であれば可へ緩和。
   - 従来チャネル: 本社・各事業場同一要件継続。

### openQuestion(要追加確認)

- GビズID・e-Govアカウント等の具体的な認証方式(OAuth2フロー種別・トークンエンドポイント・
  スコープ)の詳細: e-Gov Developer Portal で確認が必要。
- 労働者代表選任要件(過半数労働組合/過半数代表者の資格・選任手続): 社労士確認が必要。
